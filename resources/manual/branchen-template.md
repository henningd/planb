## Was ein Branchen-Template ist

Ein Branchen-Template ist ein vorbefülltes Datenpaket für ein typisches Geschäftsmodell. Wenn Sie z. B. eine **Anwaltskanzlei** sind, bringt das Template mit:

- typische **Systeme** (Kanzlei-Software, Diktiergerät, Mandantenakte, E-Mail, …);
- vorbefüllte **Notfall-Level** (z. B. Mandantenakte = kritisch, Drucker = niedrig);
- **System-Aufgaben** mit RACI-Vorbelegung;
- typische **Szenarien** (Datenpanne nach Versand an falschen Empfänger, Server-Verschlüsselung, …);
- vorgefertigte **Kommunikations-Vorlagen** (Mandanten-Info bei Datenpanne, Notfall-Aushang am Empfang).

Sie sparen damit die ersten 80 % der Tipparbeit und können sich aufs Anpassen konzentrieren statt aufs Erfinden.

## Verfügbare Branchen-Templates

| Template | Profil |
|---|---|
| Anwaltskanzlei | Mandantenakte, Diktat, Fristenkontrolle |
| Steuerberatung | DATEV, Mandanten, Fristen, Lohn |
| Arztpraxis | KV-Software, Patientenakte, Telematikinfrastruktur |
| Apotheke | Warenwirtschaft, BtM, Kühlkette, Beratung |
| Hotel | PMS, Kassensystem, Schließanlage, Reservierung |
| Einzelhandel | Kasse, Warenwirtschaft, Lager, Kartenterminal |
| Onlineshop | Shop-System, Zahlungsanbieter, Versand, Tracking |
| Heizung-/Sanitärbetrieb | Handwerker-Software, Lager, Außendienst |
| Kleine Fertigung | ERP, Maschinen-Steuerung, Lieferanten |
| Pflegedienst (ambulant) | Tourenplanung, Abrechnung, Pflegedokumentation |
| IT-Dienstleister (MSP) | Ticket-System, RMM, Kunden-Mandanten |
| Software-Unternehmen (SaaS) | Quellcode, CI/CD, Cloud-Hosting |

## So wenden Sie ein Template an

1. Im **Onboarding-Wizard** Schritt 2 erscheint die Liste der verfügbaren Templates.
2. **Klick auf ein Template** öffnet eine Vorschau: welche Systeme, welche Rollen, welche Szenarien werden angelegt.
3. **„Anwenden"** drückt das Paket in Ihren Mandanten — bestehende Daten werden nicht überschrieben, neue Einträge werden ergänzt.
4. Anschließend gehen Sie die Liste durch und passen an Ihre konkrete Realität an: Server umbenennen, Mitarbeiter zuordnen, Hotline-Nummern eintragen.

## Wann sollten Sie kein Template verwenden?

- Wenn Sie ein **sehr spezielles Geschäftsmodell** haben, das in der Liste nicht abgebildet ist — dann fangen Sie eher leer an, mit den Pflichtfeldern.
- Wenn Sie **schon vorhandene Daten** im System haben und Templates nicht mehr stark genug ergänzen, was Sie schon erfasst haben.
- Wenn Sie das Notfallhandbuch **gemeinsam mit der Geschäftsführung** durchgehen wollen und der Diskussions-Prozess wichtiger ist als die Geschwindigkeit.

## Eigene Templates anlegen (Plattform-Betreiber)

Plattform-Betreiber (Super-Admins) können unter `/admin/industry-templates` eigene Branchen-Vorlagen pflegen. Das ist ein eigener Bereich, der in dieser Anleitung nicht weiter behandelt wird — falls Sie als IT-Dienstleister Ihren Endkunden Branchenpakete anbieten wollen, sprechen Sie den Plattform-Betreiber an.

> **Tipp**: Sie können mehrere Templates kombinieren, wenn Ihr Betrieb mehrere Bereiche abdeckt — z. B. ein Hotel mit angeschlossenem Restaurant. Erst Hotel anwenden, dann das passende Gastro-Template (sofern verfügbar).
