## Worum es geht

**Abteilungen** sind die Organisations-Einheiten Ihrer Firma — z. B. Geschäftsführung, Verwaltung, Werkstatt, IT, Buchhaltung. Pro Mitarbeiter wählen Sie eine Abteilung aus dieser Liste.

Erreichbar über die Sidebar **„Einrichtung → Abteilungen"**.

## Was Sie pflegen

Pro Abteilung ein paar Felder:

- **Name** — z. B. „Werkstatt", „Verwaltung". Eindeutig pro Firma; doppelte Namen werden abgelehnt.
- **Beschreibung** — optionaler Freitext, wenn Sie kurz festhalten wollen, was die Abteilung umfasst.
- **Sortierung** — Reihenfolge, in der die Abteilungen angezeigt werden (kleinere Zahl = weiter oben).

## Wie sie verwendet werden

- **Mitarbeiter-Formular**: Beim Anlegen oder Bearbeiten eines Mitarbeiters wählen Sie die Abteilung aus einer Dropdown-Liste. „— Keine Abteilung —" ist möglich, wenn niemand zugeordnet werden soll (z. B. externe Datenschutzbeauftragte).
- **Mitarbeiter-Liste**: Filter-Drop-down oben für „Alle Abteilungen" — zeigt nur Mitarbeiter dieser Abteilung. Im Hierarchie-Modus filtert das Dropdown den DAG entsprechend.
- **System-Detailseite**: Bei Mitarbeitern wird die Abteilung neben der Position angezeigt — gut, um auf einen Blick zu sehen, welche Bereiche an einem System beteiligt sind.

## Wer was sehen darf

- **Owner und Admin** dürfen Abteilungen anlegen, bearbeiten, löschen.
- **Member** sehen die Liste read-only über das Mitarbeiter-Formular (als Drop-down-Optionen).

## Was passiert beim Löschen

Wenn eine Abteilung gelöscht wird, **bleiben die Mitarbeiter erhalten** — ihre Zuordnung wird auf „keine Abteilung" gesetzt. Sie können den Mitarbeitern dann manuell eine andere Abteilung zuweisen.

## Tipp: Branchen-Templates

Wenn Sie ein Branchen-Template anwenden, werden die mitgelieferten Abteilungen automatisch angelegt — Sie müssen sie nicht manuell eintippen.
